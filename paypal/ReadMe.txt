============ Installation Instructions ============
1. Create a database (for example, codexworld) and import the "SQL/products_payments.sql" file into this database. It will create products and payments table, and insert some demo data in the products table.

2. Open the "config.php" file in the editor ==> Specify the PayPal REST API App and database credentials.
PAYPAL_API_CLIENT_ID – Specify the REST API Client ID.
PAYPAL_API_SECRET – Specify the REST API Secret.
PAYPAL_SANDBOX – Set true for sandbox environment and false for production.

DB_HOST – Specify the database host.
DB_USERNAME – Specify the database username.
DB_PASSWORD – Specify the database password.
DB_NAME – Specify the database name.

3. Now open the "index.php" file on the browser and test the express checkout functionality.
